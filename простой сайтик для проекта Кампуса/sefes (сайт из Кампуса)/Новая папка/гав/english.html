Теперь ты можешь учиться легко и увлекательно! Главное – тренируйся регулярно и результат не заставит себя ждать.

LangBook

+ работает оффлайн
+ переводит как целые слова так и отдельные предложения
+ есть возможность проходить мини-тесты по изученному
+ есть встроенные словарики на разную тематику
- платный (около 200руб)

Reverso Context

+ приложение бесплатно
+ контекстный перевод
+ можно следить за свои прогрессом
- не работает оффлайн

Urban Dictionary

+ бесплатный
+ сленговый словарь
- не работает оффлайн
- не очень удобный в использовании

Duolingo

+ приложение бесплатно
+ максимальная удобность в изучении
+ обучение от простого к сложному

Также есть несколько приложений для общения с иностранцами и прокачки слухового навыка.
